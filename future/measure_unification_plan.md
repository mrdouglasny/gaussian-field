# Plan: Unify Gaussian Measure Definitions

## The Three Constructions

### 1. gaussian-field: `GaussianField.measure T`
**File:** `GaussianField/Construction.lean`

**Definition:** Pushforward of iid N(0,1) noise on `ℓ²` through the series limit map.
```
measure T = noiseMeasure.map (seriesLimit T)
```
where `seriesLimit T ω f = Σ' m, ⟨T f, e_m⟩ · ξ_m(ω)` (ω is a sample from the noise space).

**Key properties:**
- Takes `T : E →L[ℝ] H` (a CLM from test functions to a Hilbert space)
- Produces `Measure (Configuration E)` where `Configuration E = WeakDual ℝ E`
- Measurable space: `instMeasurableSpaceConfiguration` (cylindrical σ-algebra)
- Covariance: `C(f,g) = ⟨T f, T g⟩_H`
- IsProbabilityMeasure (proved)
- IsGaussian: `∫ exp(t·ω(f)) dμ = exp(½ t² ‖Tf‖²)` (proved)
- Requires `DyninMityaginSpace E` for the expansion

**Pros:**
- Direct construction (explicit pushforward), no axioms
- Works for ANY CLM `T : E →L[ℝ] H`, not just specific covariance forms
- Gives finite-dim and infinite-dim cases in one definition
- All properties proved (MGF, probability, moments)

**Cons:**
- Tied to the series construction (noise space + orthonormal basis)
- Requires `DyninMityaginSpace E` (the basis expansion)
- Not directly via characteristic functional (the standard physics approach)
- Doesn't use Minlos theorem

### 2. bochner: `minlos_theorem`
**File:** `Minlos/Main.lean`

**Definition:** Via Bochner-Minlos theorem: given a continuous positive-definite
normalized function Φ : E → ℂ, there exists a unique probability measure
μ on WeakDual ℝ E with `Φ(f) = ∫ exp(i·ω(f)) dμ`.

**Key properties:**
- Takes `Φ : E → ℂ` satisfying continuity, positive-definiteness, Φ(0)=1
- Requires `IsHilbertNuclear E`, `SeparableSpace E`, `Nonempty E`
- Produces `ProbabilityMeasure (WeakDual ℝ E)` (unique!)
- Construction: Kolmogorov extension of finite-dimensional marginals
- Measurable space: `MeasurableSpace.pi` on `E → ℝ`, then projected

**Pros:**
- Standard approach (Bochner-Minlos theorem)
- Works from the characteristic functional (the physics definition)
- Uniqueness is built in
- More general: any CF, not just Gaussian
- Doesn't require DyninMityaginSpace (uses IsHilbertNuclear)

**Cons:**
- Requires `IsHilbertNuclear E` (stronger than DyninMityaginSpace alone)
- Requires `SeparableSpace E`
- Construction via Kolmogorov extension (opaque, harder to compute with)
- Measurable space might differ from cylindrical σ-algebra (diamond issue)

### 3. OSforGFF (or4nge19): `constructGaussianMeasureMinlos_free`
**File:** `OSforGFF/Measure/Construct.lean`

**Definition:** Wraps bochner's Minlos theorem with the Gaussian CF:
```
Φ(f) = exp(-½ C(f,f))
```
where C is the covariance bilinear form (free propagator).

**Key properties:**
- Specialized to Gaussian: CF = exp(-½ quadratic)
- Uses `sqrtPropagatorEmbedding m` to construct the Hilbert space H and CLM T
- `gaussian_measure_characteristic_functional` from bochner provides the CF
- Returns `ProbabilityMeasure FieldConfiguration` (with FieldConfiguration = WeakDual)
- `gff_real_characteristic`: `Z[f] = exp(-½ C(f,f))` (proved from Minlos spec)

**Pros:**
- Clean API: input is covariance form, output is measure with CF
- Uniqueness for free
- Based on the standard Minlos approach
- Properties follow from the CF specification

**Cons:**
- Heavy use of `Classical.choose` (opaque, hard to compute with)
- Requires IsHilbertNuclear + Separable + Nonempty (lots of instances to supply)
- `sqrtPropagatorEmbedding` is axiomatized / uses classical choice
- Measurable space compatibility issues

## Comparison Matrix

| Feature | gaussian-field | bochner/Minlos | OSforGFF |
|---------|---------------|----------------|----------|
| Input | CLM T : E →L[ℝ] H | CF Φ : E → ℂ | Covariance C |
| Output | Measure (WeakDual ℝ E) | ProbabilityMeasure (WeakDual ℝ E) | ProbabilityMeasure (WeakDual ℝ E) |

| Uniqueness | NOT proved | PROVED (built-in) | PROVED (from Minlos) |

| Requires | DyninMityaginSpace | IsHilbertNuclear + Separable | IsHilbertNuclear + Separable |
| Explicit | Yes (pushforward) | No (Kolmogorov extension) | No (Classical.choose) |
| Computable | More (series map) | Less (extension) | Less (choice) |

## The MeasurableSpace Diamond

The CRITICAL issue: gaussian-field uses `instMeasurableSpaceConfiguration`
(cylindrical σ-algebra generated by evaluations), while bochner/Minlos uses
the product measurable space `MeasurableSpace.pi`. These are:
- **Equal** (proved as `instMeasurableSpaceConfiguration_eq_comap` in
  `ConfigurationEmbedding.lean`)
- But NOT definitionally equal in Lean

This means measures constructed by the two approaches live on different
(propositionally equal) measurable spaces, requiring transport.

## Recommendation

### Short term: Keep both, bridge them
- gaussian-field's `measure T` is used for direct construction + properties
- bochner's `minlos_theorem` is used for uniqueness
- OSforGFF already bridges them via `gff_real_characteristic`
- The bridge `MeasureUniqueness.lean` in pphi2 connects them

### Medium term: Adopt bochner/Minlos as the canonical definition
- Minlos gives uniqueness for free
- It works for non-Gaussian measures (needed for interacting P(φ)₂)
- The CF specification is the standard physics definition
- gaussian-field's series construction becomes a PROOF that the measure exists,
  not the definition itself

### Features to bring from OSforGFF to pphi2/gaussian-field
1. **`gff_real_characteristic`** — clean CF specification
2. **`gaussian_measure_characteristic_functional`** — general Gaussian CF from Minlos
3. **`IsGaussian` predicate** — characterize Gaussian measures by their CF
4. **`gff_complex_CF_covariance`** — complex CF = exp(-½ C_ℂ(f,f))
5. **`gaussianFreeField_satisfies_OS0`** — algebraic OS0 proof (no Morera needed)

### What would need to change
- gaussian-field: add a bridge theorem `measure_eq_minlos` showing the series
  construction agrees with the Minlos construction
- Resolve the MeasurableSpace diamond (probably via a canonical choice or
  by making instMeasurableSpaceConfiguration defeq to the comap)
- pphi2's `MeasureUniqueness.lean` would become simpler
